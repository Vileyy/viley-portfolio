# Viley's Personal Portfolio 🚀

A premium, modern, and highly responsive personal portfolio website designed for showcasing front-end engineering projects, professional experience, and technical skills.

Live Demo: [viley.dev](https://github.com/Vileyy) (or your deployment link)

---

## 🎨 Design & Aesthetic Highlights
* **Modern Alternating Layout**: Projects are displayed in a clean, split-row grid (alternating details and mockups) for a premium agency-like experience.
* **Premium Micro-Animations**: Built using **Framer Motion** for smooth scroll entries, floating elements, and page transitions.
* **Interactive Hero Avatar**: Features interactive rotating orbital lines, glowing active badges, and floating technology icon badges (React, TypeScript, Next.js) that animate gently.
* **Official Brand Icons**: All technology badges and contact cards are styled with their official, colored brand logos from `react-icons`.
* **Dark Mode Compatible & Glassmorphism**: Tailored HSL colors, clean glassmorphic components, and drop-shadow depths.

## 🚀 Getting Started (Local Development)

This project uses **pnpm** as its primary package manager. Follow the steps below to run it locally:

### 1. Clone the repository
```bash
git clone https://github.com/Vileyy/viley-portfolio.git
cd viley-portfolio
```

### 2. Install dependencies
```bash
pnpm install
# or if you use npm:
# npm install
```

### 3. Run the development server
```bash
pnpm dev
# or:
# npm run dev
```
Open your browser and navigate to `http://localhost:5173`.

### 4. Build for production
```bash
pnpm build
# or:
# npm run build
```
